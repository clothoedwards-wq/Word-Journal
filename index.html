Term 1: Technology

My Definition: The application of knowledge, skills, tools, and techniques to solve problems, create new possibilities, or improve the way tasks are performed. It encompasses everything from simple tools and machines to complex systems like computers, the internet, and artificial intelligence. Technology is both a product of human ingenuity and a driver of innovation, shaping how we interact with the world and each other.

Synonyms:Tools, Equipment, Machinery, Invention, Media, nnovation, Devices, Applied Science, Gadgets

Antonyms:Natural Processes, Labor, Tradition, Primitive Methods, Handcraft.

Source

Term 2: Computer

My Definition: a machine that processes and stores information. It follows instructions from programs to perform tasks like calculations, playing games, or connecting to the internet. Computers have physical parts (hardware) and programs (software) that work together to get things done.

Synonyms:PC (Personal Computer), Laptop, Desktop, Workstation, Server, Processor, Terminal, Mainframe, Device, System, Tech

Antonyms: Manual Tools, Analog Tools, Non-electronic Devices, Typewriter, Books, Non-digital Equipment.

Source

Term 3: HTML

My Definition: a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on web pages.

Synonyms:
  • Markup language
  • Web markup
  • Document markup language
  • Web page language
  • Tag-based language

Antonyms: Plain text, Binary code, Machine code, Back-end processing language, Styling language.

Source

Term 4:HTML Tag


My Definition: special keyword used in HTML (HyperText Markup Language) to create and structure content on a web page.

Synonyms:
  • HTML elements
  • Markup elements
  • Tag elements
  • Web page tags

Antonyms: head ↔ body, header ↔ footer, title ↔ meta.

Term 6:Markup Language


My Definition: a system for annotating a document with tags or codes that define its structure, formatting, and content, rather than its logic.

Synonyms:
  • Tagging language
  • Markup system
  • Text markup system
  • Annotation language
  • Document markup language

Antonyms: Programming language, Scripting language, Plain text.

Term 7: coding

My Definition: the process of writing instructions (code) in a specialized language that tells a computer, application, or device to perform specific tasks, such as creating software, websites, or automating processes.

Synonyms: Software Development, Software Engineering, Application Development/Programming, Scripting, Implementing, and Computer Science

Antonyms: Refactoring, Commit, Software, Structured, Soft Coding

Term 8: compiler

My Definition: a software program that translates human-readable source code, written in a high-level programming language, into low-level machine code, bytecode, or assembly language.

Synonyms: assembler, compilator, combiner, and gatherer

Antonyms: disperser, scatterer, distributor
